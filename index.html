On this page I will now explain some of the biggest challenges that developers face when trying to design websites/applications for mobile devices.

Challenges: Why it's a challenge: How it could be solved:
Standardisation This is one of the biggest problems for web developers as there are no set standards for web development on mobile devices. I believe that the best way to fix this problem would be to have a selection of the biggest web developers/companies to set a standard for web development that developers can work to.
Smaller Displays Another of the biggest issues of web development is that developers are required to perform a lot of extra work to ensure that the work they're creating can be accessed from all devices. The only true solution to this problem is when mobile device screen sizes eventually become standardised.
Speed The issue with the speed can be both the processing power of the mobile device that it's running on and the speed in which the phone can retrieve the data from the internet. This can be a major problem as some web pages could take a very long time to load. porn
